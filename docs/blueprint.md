# **App Name**: Grounded Gemini

## Core Features:

- Search Query Input: Accepts user input for search queries and leverages the Gemini API with Google Search grounding tool to formulate a grounded response.
- Grounded Response Generation: Utilize the Gemini API to process user queries and provide accurate answers based on real-time web content.
- Response Display: Display the generated grounded response in a clean and readable format.
- Source Citation Display: Visually represent web sources used in the grounding process.
- Display Search Queries: Displays the web search queries that the LLM "tool" constructed to generate the final answer.
- Loading Status: Shows a "loading" status to communicate that the app is waiting on the LLM's call to the Gemini API.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and intelligence.
- Background color: Light gray (#F5F5F5) for a clean, neutral backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and citations.
- Body font: 'Inter', a sans-serif font, for readability.
- Headline font: 'Space Grotesk', a sans-serif font, to emphasize modernity and precision.
- Use simple, geometric icons to represent different categories of search results.
- Use a clean and modern layout to emphasize search results. Prioritize readability and ease of use.
- Subtle loading animations while the grounded response is being generated.
